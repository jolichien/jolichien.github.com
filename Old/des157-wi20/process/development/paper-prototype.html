Final Project - Paper Prototypes

For my final project, I've decided to create a sort of interactive-storytelling-educational infographic that warns about internet negativity. So for my paper prototypes, I tried to represent some of the interactive aspects that I want to include. The downward arrow suggests that the user should scroll downwards which leads to the next panel.

For this interactive portion, the idea is that the user can click on the negative comments to make them go away, but only temporarily.

For this interactive portion, the user can hover over the masks to reveal a face underneath.

For this interactive portion, the lightbulb can be clicked on to turn the light on. This portion is meant to be a great contrast from the initial look of the page.
